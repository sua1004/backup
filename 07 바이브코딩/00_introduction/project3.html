프로젝트 개요

Vue.js와 Node.js를 결합하여 개발한 풀스택 웹 애플리케이션입니다. 프론트엔드와 백엔드를 모두 구현하여 완전한 웹 서비스를 제공합니다.

RESTful API를 통한 데이터 통신과 실시간 업데이트 기능을 구현했습니다.

주요 기능

  • ✓ RESTful API 구현
  • ✓ 데이터베이스 연동
  • ✓ 인증 및 권한 관리
  • ✓ 실시간 데이터 동기화

기술 스택

Vue.js 3 Node.js Express.js MongoDB JWT
