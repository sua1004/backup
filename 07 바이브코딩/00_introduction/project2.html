JavaScript HTML5 CSS3

프로젝트 개요

순수 JavaScript를 활용하여 개발한 인터랙티브 웹 애플리케이션입니다. 바닐라 JavaScript의 강력함을 활용하여 프레임워크 없이도 현대적인 웹 경험을 제공합니다.

DOM 조작과 이벤트 핸들링을 중심으로 사용자와의 상호작용을 구현했습니다.

주요 기능

  • ✓ 동적 콘텐츠 렌더링
  • ✓ 사용자 인터랙션 처리
  • ✓ 로컬 스토리지 활용
  • ✓ 반응형 레이아웃

기술 스택

JavaScript ES6+ HTML5 CSS3 LocalStorage
